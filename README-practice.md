✅ 구현 체크리스트 (기능 단위)
- [] 입력 처리
- 사용자에게 근무 배정의 월과 시작 요일을 받는다.
비상 근무를 배정할 월과 시작 요일을 입력하세요> 5,월
-> 받은 문자를 spilt으로 나누기 

- [] 비상 근무를 배정할 월과 시작 요일의 입력 값이 올바르지 않은 경우, '비상 근무를 배정할 월과 시작 요일'부터 다시 입력받는다.

- [] 월별 날짜와 휴일 정보에 관한 내용을 constant.js 로 상수로 분리해둔다. 

- [] Day.js로 입력받은 월과 시작 요일을 매칭해 준다.
입력 받은 월은 constant.js를 사용하여 시작 며칠까이 있는 지를 확인하여 for문을 돈 다음, 순서대로 시작 요일을 매칭해 준다. 

-> 나오게 되는 결과는 몇월인 지와, 1일부터 요일에 대한 정보이다. 


- [] 2.평일 비상 근무 순번을 받는다. 
평일 비상 근무 순번대로 사원 닉네임을 입력하세요> 
-> 받은 문자를 spilt으로 나누기 


- [] 휴일 비상 근무 순번을 받는다. 
휴일 비상 근무 순번대로 사원 닉네임을 입력하세요>
-> 받은 문자를 spilt으로 나누기 

- [] 평일 순번 혹은 휴일 순번의 입력 값이 올바르지 않으면, '평일 순번'부터 다시 입력을 받아야한다.

- [] 받은 정보를 바탕으로 1일이 무슨 요일인지에 따라서 사람을 배치한다. 만약 월요일이라면 평일 순번 5명과 주말 순서 2명. 
화요일이라면 평일 순번 4명과 주말 순서 2명 -> 이후 다시 평일 순번 5명과 주말 순서 2명의 순이 된다. 
맨 앞의 평일 순번이 달라지게 되는 것.

- [] 평일이면서 법정 공휴일에만 요일 뒤에 (휴일)을 표기해야한다. 

- [] 해당 내용을 출력 후 종료된다.
